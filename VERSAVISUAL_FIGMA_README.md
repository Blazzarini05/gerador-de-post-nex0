# VersaVisual Brand System - Figma Integration

Este arquivo contém todo o sistema de identidade da VersaVisual preparado para uso no Figma.

## 📁 Arquivos Disponíveis

### 1. `versa_visual_brand_system.json`
Arquivo completo com todas as informações da marca, incluindo:
- Identidade e pilares da VersaVisual
- Paleta de cores completa
- Sistema tipográfico
- Tom de voz e frases de referência
- Estratégia de conteúdo
- Templates disponíveis

### 2. `versa_visual_variables.css`
Variáveis CSS prontas para importação no Figma Variables, contendo:
- Cores da marca como variáveis CSS
- Sistema tipográfico
- Espaçamentos e bordas
- Sombras e componentes base

## 🎨 Como Importar no Figma

### Método 1: Figma Variables (Recomendado)
1. Abra seu arquivo no Figma
2. Vá em **Assets** → **Variables**
3. Clique em **Import** → **From file**
4. Selecione `versa_visual_variables.css`
5. As variáveis serão importadas automaticamente

### Método 2: Manual
1. Copie as cores do arquivo JSON:
   - VV Black: #0A0A0A
   - VV White: #FFFFFF
   - VV Teal Accent: #5B9BAF
   - VV Light Text: #E0E0E0
   - VV Section Label: #7AABB8

2. Configure a tipografia:
   - Display: Bebas Neue / Barlow Condensed (fallback)
   - Body: System fonts (ui-sans-serif)

## 💡 Dicas de Uso

1. **Consistência**: Sempre use as cores definidas para manter a identidade
2. **Hierarquia**: Use Bebas Neue para títulos principais
3. **Espaçamento**: Mantenha múltiplos de 4px para alinhamento
4. **Tom de voz**: Consulte as frases de referência antes de criar conteúdo
